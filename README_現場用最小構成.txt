【現場で"今すぐ使える"最小構成の使い方】

---

1. サーバーを使う場合（Node.js/Express）

■ 準備
- version2フォルダ内に simple_server.js があることを確認してください。

■ 手順
1. コマンドプロンプトを開きます。
2. 下記コマンドを順番に入力します。

   cd "C:\Users\user\OneDrive\Desktop\QRコード認証\version2"
   node simple_server.js

3. "Simple server started on http://localhost:3000"と表示されたらサーバー起動完了です。
4. ブラウザで下記URLにアクセスします。
   http://localhost:3000/admin_login.html
5. ログイン情報を入力します。
   - ユーザー名：General administrator
   - パスワード：OGAWA SEIJI
6. 管理画面が表示され、各種操作が可能です。

---

2. サーバー不要（ローカル認証のみ）の場合

■ 手順
1. version2フォルダ内の admin_login.html をダブルクリックしてブラウザで開きます。
2. 上記と同じログイン情報で管理画面に入れます。

---

3. 注意点
- この構成はPC単体での運用や一時的な現場テストに最適です。
- 多人数・院内LAN連携・操作ログ等は本格運用時にサーバー方式へ移行してください。
- パスワードはHTML/JSに記述されているため、セキュリティは最低限です。

---

4. トラブル時
- サーバーが起動しない場合は、コマンドプロンプトのエラーメッセージを確認してください。
- ブラウザで画面が表示されない場合は、URLやファイルパスを再確認してください。

---

5. 本格運用への移行
- 現場の運用が固まったら、サーバー・DB・多人数管理・セキュリティ強化を段階的に導入してください。
